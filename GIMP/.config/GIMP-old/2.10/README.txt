To install:
------------------------------------------------------------


GENERAL PATHS (including support files):

	/gimp/2.0/gimpressionist/*
	/gimp/2.0/gradients/*
	/gimp/2.0/patterns/*
	/gimp/2.0/scripts/*
	/gimp/2.0/scripts/images/*
	
	
	
	
Put the scripts in :

	Windows Vista/7/8/10)
	C:\Program Files\GIMP 2\share\gimp\2.0\scripts
	or
	C:\Users\YOUR-NAME\AppData\Roaming\GIMP\2.10\scripts
    
	
	Linux
	/home/yourname/.config/GIMP/2.10/scripts
	
	
	Linux system-wide
	/usr/share/gimp/2.0/scripts





Put the Gimpressionist presets in :

	Windows Vista/7/8/10
	C:\Program Files\GIMP 2\share\gimp\2.0\gimpressionist\preset
	or
	C:\Users\YOUR-NAME\AppData\Roaming\GIMP\2.10\presets
	
	
	Linux
	/home/yourname/.config/GIMP/2.10/gimpressionist/presets
	
	
	Linux - system-wide
	/usr/share/gimp/2.0/gimpressionist/Presets 




Put the Gradients in :

	Windows Vista/7/8/10
	C:\Program Files\GIMP 2\share\gimp\2.0\gradients
	or
	C:\Users\YOUR-NAME\AppData\Roaming\GIMP\2.10\gradients
	
	
	Linux
	/home/yourname/.config/GIMP/2.10/scripts
	or
	Linux - system-wide
	/usr/share/gimp/2.0/gradients
	
	


Put the patterns in :

	Windows Vista/7/8/10
	C:\Program Files\GIMP 2\share\gimp\2.0\patterns
	or
	C:\Users\YOUR-NAME\AppData\Roaming\GIMP\2.10\patterns
	
	
	Linux
	/home/yourname/.config/GIMP/2.10/patterns
	or
	Linux - system-wide
	/usr/share/gimp/2.0/patterns




Put the images in :

	Windows 7/10
	C:\Program Files\GIMP 2\share\gimp\2.0\scripts\images
	or
	C:\Users\YOUR-NAME\AppData\Roaming\GIMP\2.10\scripts\images
	
    
	Linux
	/home/yourname/.config/GIMP/2.10/scripts/images
	or
	Linux system-wide
	/usr/share/gimp/2.0/scripts/images

About:
------------------------------------------------------------
gimp_scripts-2.10 are a fixed/hacked collection of scripts for 
use in the new GIMP. Originally for 2.4, but things continue 
to evolve and break scripts... so on I go. 

If a script I wrote or played with gives me errors in use, I 
get them running and include them in the scripts package. 
I am not the original author for most of these scripts, but 
I believe they all let you redistribute (GPL-like).

So use them, pass them around or make them better.
It's all good.

Enjoy.
